# Solar Salvager

A small browser arcade game built with plain HTML, CSS, and JavaScript.

## Play online

The game is intended to be shared as a static website once GitHub Pages is enabled for this repository.

## Run locally

Open `index.html` in a browser.

## Controls

- `WASD` or arrow keys to move
- Hold and drag on the game area for mouse or touch steering
- `Space` or the on-screen shockwave button to trigger the pulse when charge reaches `100%`

## Goal

Stay alive for 90 seconds. Enemies drop scrap capsules that refill your pulse charge.
